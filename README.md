**Автор:** [Сахабиева Диана (из pragmata)]

## 📖 Описание
Графическое приложение для хранения, фильтрации и управления личной коллекцией фильмов. Разработано на Python с использованием стандартной библиотеки tkinter. Данные сохраняются и загружаются в формате JSON.

## 💻 Требования
- Python 3.8 или выше
- Встроенные модули: tkinter, json, os, messagebox

## 🚀 Установка и запуск
1. Скачайте или клонируйте репозиторий.
2. Запустите в терминале: `python movie_library.py`
3. Для сохранения данных используйте кнопку «💾 Сохранить в JSON», для загрузки — «📂 Загрузить из JSON».

## ⚙️ Функционал
- ✅ Добавление фильмов с валидацией полей
- ✅ Отображение коллекции в таблице
- ✅ Фильтрация по жанру и году выпуска
- ✅ Импорт/экспорт данных через JSON
- ✅ Поддержка кириллицы (UTF-8)

## 🧪 Примеры использования (Тесты)
| № | Сценарий | Действия | Ожидаемый результат |
|---|----------|----------|---------------------|
| 1 | Успешное добавление | Заполнить все поля → «Добавить фильм» | Запись в таблице, поля очищены, сообщение об успехе |
| 2 | Ошибка года | Ввести Год: 199x → «Добавить фильм» | Предупреждение: «Год должен быть целым числом» |
| 3 | Ошибка рейтинга | Ввести Рейтинг: 11.5 → «Добавить фильм» | Предупреждение: «Рейтинг должен быть числом от 0 до 10» |
| 4 | Пустые поля | Оставить «Название» пустым → «Добавить фильм» | Предупреждение о обязательных полях |
| 5 | Фильтрация | Выбрать жанр → «Применить фильтр» | В таблице только фильмы выбранного жанра |
| 6 | Сброс фильтра | Нажать «Сбросить» | Отображение полного списка фильмов |
| 7 | Сохранение/Загрузка | Сохранить → закрыть → открыть → загрузить | Данные корректно восстанавливаются из movies.json |
