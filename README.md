## 📝 Описание
GUI-приложение для хранения, фильтрации и управления личной коллекцией фильмов. Разработано на Python с использованием стандартной библиотеки `tkinter`. Данные сохраняются и загружаются в формате JSON.

## ⚙️ Функционал
- ✅ Добавление фильмов с автоматической валидацией полей
- ✅ Отображение коллекции в удобной таблице
- ✅ Фильтрация по жанру и году выпуска
- ✅ Сохранение и загрузка базы данных в/из `movies.json`
- ✅ Поддержка кириллицы и UTF-8
